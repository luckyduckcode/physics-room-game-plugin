# Physics Room Splats — Unreal stub

This folder contains a minimal Unreal Engine plugin stub that documents where
to add a PLY splat loader for runtime import. It's a scaffold for teams that
want to implement a native importer (C++) or a Blueprint wrapper.

Files
- `SplatPlugin.uplugin` — plugin manifest
- `Source/SplatPlugin/` — suggested source layout for a runtime module

Notes
- This is a non-compiling scaffold; add your engine-version-specific module
  and implementation as needed (UE5 module rules, target settings, etc.).
