Physics Room Splats Godot Add-on
================================

This minimal add-on provides an editor button to help reload point-cloud
exports (PLY/JSON) produced by the `physics_engine` exporter. It's a
scaffold to be extended by game projects — add your runtime loader script
that watches a path and updates a `MultiMeshInstance` or `ShaderMaterial`.

Install:
- Copy the `addons/physics_room_splats` folder into your Godot project
- Enable the add-on via Project -> Project Settings -> Plugins
